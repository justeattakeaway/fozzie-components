Buttons allow users to take actions, and make choices, with a single tap. They are typically placed throughout UI, in places like:

Dialogs
Modal windows
Forms
Cards
Toolbars


| Button type | Purpose |
| : | : |
| Primary | Have the most emphasis, as they use a color fill. They are used for the principal call to action on the page.|
| | Use only one per page |
| | Use for the most important action on the page |
| | Save, Submit, Next etc |
| Secondary | For secondary actions on each page. |
| | Can be used on its own, or in conjunction with a primary button |
| | Use for non-essential actions |
