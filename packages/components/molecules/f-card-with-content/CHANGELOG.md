v0.2.0
------------------------------

### Added
- Heading
- Description
- Primary and secondary buttons
  - These emit events like `primary-button-click`
  - Expect objects containing the properties:

    - (Optionally) `href`/`to` for rendering the button as an anchor or router-link.
- Named slot for icon
- Storybook story
